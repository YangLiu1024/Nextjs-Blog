# 概述
Vue 是一个用于构建用户界面的渐进式 JavaScript 框架。 Vue 的核心库只关注视图层，当数据改变时，视图会自动更新。

## 模板语法
Vue 使用了基于 HTML 的模板语法，允许开发者声明式的将 DOM 绑定至底层组件实例的数据。在底层实现上，Vue 会将模板编译成虚拟的 DOM 渲染函数，这个就和 React 的 render 函数很类似。  
如果你熟悉 JS, 偏爱 JS 的原始力量，也可以不使用模板语法，而是和 React 一样，直接写 render 函数，当然，你也可以在 Vue 里 使用 JSX 语法。  
在数据改变时，Vue 能够智能的计算出最少需要重新渲染的组件，并且将 DOM 操作次数减到最少。

该例和上例的区别就在于，我们不再使用 HTML 里挂载元素的内容作为模板，而是通过 *template* 选项自定义模板。 

## Vue & React
Vue 和 React 有很多相似之处，都提供了
* 虚拟 DOM 树
* 响应式和组件化的视图
* 都专注于核心库，而将其它功能，比如路由和全局状态管理交给其它相关库

在性能方面，React 和 Vue 都是非常快的
