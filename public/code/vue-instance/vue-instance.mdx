# Vue 实例
每个 Vue 应用都是通过用 Vue 函数创建一个 Vue 根实例，以及可选的，可复用的组件树组成，每一个 Vue 组件都是 Vue 实例。 

## Vue 实例选项 
在创建 Vue 实例时，可以传入一个选项对象，在该选项对象里进行配置来创建你想要的行为。  
```js
let vm = new Vue({
    el: "#root", //指定挂载的 DOM 元素, Vue 实例渲染出来的结果，将会替换指定元素的内容
    template: `<div></div>`, //该实例使用的模板，会在此模板的基础上进行渲染, 注意该模板只能有一个根元素
    data: {},//该实例的数据对象，对象里每一个元素都自动变成响应式的, 如果是组件，data 选项必须为函数
    computed: {},//实例的计算属性，会根据依赖自动更新，和 React 的 useMemo 类似
    methods: {},//实例的方法，一般用在事件响应中，方法实现中 this 会自动绑定到 vue 实例
    components: {},//声明该组件使用的其它组件
    props: {},//显式定义该组件可以接收的参数，由父组件传入
    render: h => {}//渲染函数，如果不想使用 template 语法，可以直接写渲染函数, 这个就和 React 的 render 函数类似，只是有自己的一些规则
    //还有一些其它的选项，比如生命周期钩子函数，beforeCreate, created, 等等   
})
```
